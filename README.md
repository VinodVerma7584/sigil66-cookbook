# Sigil66 Cookbook

Base cookbook for local builds and labs

## Supported Platforms

- Ubuntu 16.04 

## Attributes

None.

## Usage

### sigil66::default

Include `sigil66` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[sigil66::default]"
  ]
}
```
